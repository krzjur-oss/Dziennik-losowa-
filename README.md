# 📒 Dziennik Losowania

Aplikacja PWA do losowania uczniów z dziennika klasowego. Działa w przeglądarce na tablecie i telefonie z Androidem — bez instalacji, offline.

## ✨ Funkcje

- Wiele klas z osobnymi pulami numerów (1–30)
- Losowanie 2 numerów dziennie bez powtórzeń aż do wyczerpania puli
- Historia losowań z datami dla każdej klasy
- Eksport / Import danych (kopia zapasowa)
- Działa w pełni offline po zainstalowaniu

## 📱 Instalacja na Androidzie

1. Wejdź na: **https://krzjur-oss.github.io/dziennik-losowania**
2. W Chrome naciśnij menu ⋮ → **"Dodaj do ekranu głównego"**
3. Gotowe — aplikacja działa jak natywna, bez paska przeglądarki

## 🔒 Prywatność

Wszystkie dane (klasy, historia losowań) przechowywane są **wyłącznie lokalnie** w przeglądarce urządzenia. Żadne dane nie są wysyłane na serwer.

## 🛠️ Technologie

- Vanilla HTML/CSS/JS — zero zależności
- PWA (Progressive Web App) z Service Workerem
- localStorage do przechowywania danych
